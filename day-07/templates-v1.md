# Reusable Prompt Templates — v1

### 1. Summarize
```
Summarize {text} for {audience}, in {output_format}.
```
**Filled example:** Summarize this incident log for a junior SOC analyst, in 3 bullet points.

### 2. Classify
```
Classify {email} as {category_list}, and explain your reasoning in 1 sentence.
```
**Filled example:** Classify this email as Phishing / Legitimate / Spam, and explain your reasoning in 1 sentence.

### 3. Explain
```
Explain {concept} to {audience}, using {output_format}.
```
**Filled example:** Explain zero-day vulnerability to a non-technical manager, using a short analogy.
